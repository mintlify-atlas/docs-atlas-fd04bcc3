## Welcome to gurl

gurl is a modern HTTP client built in Go that combines the power of curl with additional features designed for today's API-driven development. Inspired by curl but enhanced with JSON formatting, progress tracking, and developer-friendly defaults.

## What Makes gurl Different?

- **Intuitive Syntax**: Cleaner command structure compared to traditional curl
- **Built-in JSON Handling**: No need for external tools like jq for JSON formatting
- **Download Manager**: Integrated download command with resume and retry capabilities
- **Modern Defaults**: Sensible defaults for API development workflows
- **Fast & Reliable**: Built in Go for performance and cross-platform compatibility
