## Overview

The `version` command displays the current version of gurl installed on your system.

## Syntax

```bash
gurl version
```

Or using the shorthand flag:

```bash
gurl -V
```

## Examples

### Display Version

```bash
gurl version
```

**Output:**
```
gurl version 1.0.1
```

### Using Shorthand Flag

```bash
gurl -V
```

**Output:**
```
gurl version 1.0.1
```

## Usage

This command is useful for:

- Checking which version of gurl is installed
- Verifying successful installation or upgrade
- Reporting version information when filing bug reports
- Ensuring compatibility with documentation or scripts

## Notes

- The version follows semantic versioning (major.minor.patch)
- Both `gurl version` and `gurl -V` produce identical output
- No additional flags or arguments are required
